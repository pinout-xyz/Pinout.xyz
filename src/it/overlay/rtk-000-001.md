#RTK Motor Controller

###Una maniera facile e veloce per controllare dei motori dal tuo Raspberry

```python
from gpiozero import RyanteckRobot
from time import sleep

robot = RyanteckRobot()

while True:
    robot.forward()
    sleep(5)
    robot.left()
    sleep(1)
    robot.backward()
    sleep(5)
    robot.right()
    sleep(1)
    robot.stop()
    sleep(1)
```
