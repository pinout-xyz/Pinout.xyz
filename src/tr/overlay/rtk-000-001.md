#Ryanteck Motor Kontrol Kartı

###Bu kart, Raspberry Pi'nizde motor sürmek için hızlı ve de kolay bir yol sağlamakta.


```python
from gpiozero import RyanteckRobot
from time import sleep

robot = RyanteckRobot()

while True:
    robot.forward()
    sleep(5)
    robot.left()
    sleep(1)
    robot.backward()
    sleep(5)
    robot.right()
    sleep(1)
    robot.stop()
    sleep(1)
```
