# Skywriter HAT

Skywriter HAT X, Y ve Z ekseninde parmağınızın pozisyonunu algılamanızı, ve de bu değerleri Python Scriptinizde kullanmanızı sağlar.

```bash
curl -sS https://get.pimoroni.com/skywriter | bash
```
