# Fan SHIM

Add a 30mm fan to your Pi 4 keeping it around 20 degrees cooler. Fan SHIM uses a friction-fit header so it can be pushed into place.

## Features

* Surprisingly quiet
* 4,200 RPM fan
* No soldering required
* Friction-fit header
* Single APA-102 RGB LED
* Tactile switch
